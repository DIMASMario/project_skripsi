# Instruksi Logo Desa

## Lokasi File Logo
Simpan logo desa dengan nama: **`logo-desa.png`**

📁 Path lengkap: `c:\wamp64\www\project-skripsi\desa_tanjungbaru\public\images\logo-desa.png`

## Spesifikasi Logo
- **Format**: PNG (dengan background transparan lebih baik)
- **Ukuran**: Minimal 200x200 pixels, maksimal 500x500 pixels
- **Rasio**: Persegi (1:1) atau mendekati persegi
- **Kualitas**: Resolusi tinggi untuk tampilan yang tajam

## Logo yang Digunakan
Logo **Desa Benteng Pancasila** - Kabupaten Subang dengan karakteristik:
- Bentuk perisai hijau dengan border kuning
- Motif bangunan/rumah di tengah
- Bintang di bagian atas
- Text "KARYA UTAMA SATYA NEGARA"

## Cara Menyimpan Logo
1. Simpan gambar logo yang telah diberikan
2. Rename file menjadi: **logo-desa.png**
3. Copy/paste ke folder: `public/images/`
4. Pastikan nama file tepat (huruf kecil, dengan tanda strip)

## Penggunaan
Logo ini akan muncul di:
- ✅ Halaman Login Warga
- ✅ Halaman Registrasi Warga
- ✅ Halaman Login Admin (jika sudah diupdate)

## Troubleshooting
Jika logo tidak muncul:
1. Cek nama file: harus **logo-desa.png** (case-sensitive di Linux)
2. Cek lokasi: harus di folder `public/images/`
3. Cek permission file (di Linux): `chmod 644 logo-desa.png`
4. Clear browser cache: Ctrl + F5
5. Cek console browser untuk error loading image
